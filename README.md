# Pneumonia Chest X-Ray Classification

## Overview
This project uses Deep Learning and Convolutional Neural Networks (CNNs) to classify chest X-ray images as **Pneumonia** or **Normal**. The goal is to support early pneumonia detection through automated medical image analysis.

## Features
- Chest X-ray image classification
- Binary prediction: Normal or Pneumonia
- Image preprocessing and augmentation
- CNN-based deep learning model
- Model training and evaluation
- Performance visualization

## Dataset
The project uses a publicly available Chest X-Ray Images (Pneumonia) dataset containing labeled X-ray images of normal and pneumonia cases.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- OpenCV
- Jupyter Notebook
